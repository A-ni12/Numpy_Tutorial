# Numpy_Tutorial
"Beginner to Intermediate NumPy Tutorial with Examples in Python."

This repository contains a complete NumPy tutorial for students learning Data Science and Python.

## Topics Covered
1. Introduction to NumPy
2. NumPy Arrays
3. Array Operations
4. Indexing and Slicing
5. Mathematical Functions
6. Statistics using NumPy

## Requirements
- Python
- NumPy
- Jupyter Notebook

## Installation
pip install numpy
